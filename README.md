# MTG MTProto Proxy — быстрый деплой

Автоматическая установка MTProto прокси для Telegram на базе [MTG v2](https://github.com/9seconds/mtg) с Fake TLS (домен `ya.ru`, порт `443`).

**Требования:** Ubuntu 24.04 или новее, root доступ.

---

## Быстрый старт

```bash
git clone https://github.com/Ddementef/MTProtoAuto.git
cd MTProtoAuto
bash mtg.sh
```

При первом запуске выберите пункт **1 — Установить прокси**. Скрипт:
- установит Docker (если не установлен)
- сгенерирует секрет Fake TLS
- запустит контейнер
- выведет готовую ссылку `tg://proxy?...` для добавления в Telegram

## Детали

| Параметр | Значение |
|---|---|
| Образ | `nineseconds/mtg:2` |
| Порт | `443` |
| Fake TLS домен | `ya.ru` |
| Файлы прокси | `/opt/mtg/` |

Прокси запускается через `docker compose` с `restart: unless-stopped` — автоматически стартует при перезагрузке сервера.
